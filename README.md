# 🛡️ Agentic AI Screen Time Guardian
An AI-powered screen time management system that uses computer vision and LLM agents to protect children's eye health.

## Features
- 👁️ Real-time blink detection using OpenCV & MediaPipe
- 🧠 AI Agent with memory for smart decisions
- 🔒 Automatic screen locking
- 📱 Parent alerts via SMS
- 📊 Web dashboard for monitoring

## Tech Stack
- Python 3.11+
- OpenCV, MediaPipe
- LangChain, ChromaDB
- FastAPI, Flask
- Docker
